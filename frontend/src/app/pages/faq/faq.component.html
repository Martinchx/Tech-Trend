Frequently Asked Questions

What is the project about?

Tech-Trend is an e-commerce platform where users can authenticate, browse, filter, search for products, manage their shopping carts, and place orders. Administrators oversee users and orders, and handle the uploading of categories and products.

What technologies did you use for the project?

I used TypeScript as the main language. In the frontend, I used Angular, while for the backend I used Nest. The database used is PostgreSQL, managed using the ORM Prisma. Authentication was implemented using JSON Web Tokens (JWT).

What cloud services did you use for the project?

At the moment I only integrated Cloudinary to host the images of the products, I did not opt for AWS S3 because I want to avoid billing problems, and in the case of sending transactional mails, it is in my plans to add a service like Brevo or Resend.

How can I test the administrative dashboard?

For security reasons I don't have public credentials for an administrator user, what you would have to do is to go to the project repository and create one in your database.

How can I test stripe's payment functionality without using my real account?

Stripe itself provides test cards and accounts. You can find detailed information and test credentials in their documentation.

How can I contribute to the project?

You can contribute by going to the project repository and opening a pull request, I'll keep an eye out for any changes/improvements to the site.

Why didn't you add an email validation at the time of registration?

Mainly because mailing systems require a domain for managing transactional emails, and since this is a personal project, I don't plan to buy one. Using my own account could lead to spam issues and potential bans.
